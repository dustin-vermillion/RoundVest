# RoundVest
An AI-driven micro-investing platform that rounds up daily UPI spends and dynamically routes funds to NIFTY 50 or Digital Gold based on real-time market sentiment.

Working Principle :

Track : Securely monitors UPI spends and rounds them up to the nearest adjustable value into a virtual wallet.

Analyse : AI scans global financial news to generate a real-time "Market Mood Score.“

Invest : At the ₹100 threshold, it auto-routes funds into NIFTY 50 (stable market) or Digital Gold (volatile market).

Investment Flexibility : User has multiple investment options to select from.
